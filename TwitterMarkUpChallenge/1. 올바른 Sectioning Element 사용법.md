# Sectioning elements의 공통적인 특징

### section tag = 단원

### (section, article, nav, aside)

- 단원에는 주제 or 제목이 있다.
- 즉! heading태그가 반드시 존재해야 한다!

  ```
  <section>
    <h1>제목</h1> ===> !!! 필수 !!!
    <p>~~~</p>
  </section>
  ```

  ```
  <nav>
    <h1>제목</h1> ===> !!! 필수 !!!
    <ul>
      <li></li>
      <li>
        <a harf="#"></a>
      </li>
      <li></li>
    </ul>
  </nav>
  ```
